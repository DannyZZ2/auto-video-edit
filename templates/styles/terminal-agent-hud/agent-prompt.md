# Terminal Agent HUD Agent Prompt / 终端 Agent HUD 提示

Use this style for prompt, command, generation, verification, export, skill packaging, and AI agent execution moments.

用于 prompt、命令、生成、校验、导出、skill 打包和 AI Agent 执行片段。

Rules:

1. Use short terminal or agent cards, not long real terminal dumps.
2. Keep 2-4 rows such as `ANALYZE`, `GENERATE`, `VERIFY`, `EXPORT`.
3. Purple can imply model/agent, green means complete, amber means check pending.
4. Use short type-in, status dot pulse, and completed row check. Avoid full-screen code rain.
