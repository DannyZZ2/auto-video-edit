# Diagnostic Glass Cards Agent Prompt / 诊断玻璃卡片提示

Use this style for lightweight overlays, soft key-point cards, side prompts, feature summaries, and ending skill/file callouts.

用于轻量重点弹窗、侧边提示、功能摘要和结尾 skill/文件提示。

Rules:

1. Use translucent cards with blur, rim highlight, soft shadow, and high-contrast text.
2. Keep semantic teal/blue/amber accents small: pills, labels, side rails, or icon glows.
3. One local shine is allowed on entrance. Do not loop full-screen shine or scan effects.
4. Keep cards airy, readable, and outside subtitle and face/mouth safe zones.
