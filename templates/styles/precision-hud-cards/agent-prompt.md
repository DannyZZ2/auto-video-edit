# Precision HUD Cards Agent Prompt / 精密 HUD 卡片提示

Use this style for precise status cards, metrics, comparison, workflow steps, and compact engineering-like explanations.

用于状态卡、指标拆解、对比结论、流程步骤和精密工程感解释。

Rules:

1. Use tight grids, 1px semantic borders, mono metadata, and 2-4 structured rows.
2. Blue means active path, green means pass, amber means caution.
3. Every line, tick, and progress bar must explain content. No decoration-only dashboard.
4. Use row stagger, border pulse, and numeric/progress growth tied to keyword cues.
5. Keep the card outside face, mouth, gestures, and subtitle safe zones.
